Air Quality Dashboard

About This Project

Project Overview

This project is a web application that provides historical air quality data for various cities. Users can enter a city name and optionally select a date range to view PM2.5 pollution trends. Users may also discuss their observations in the comments section.

Technologies Used

  • OpenAQ API - For real-time and historical air quailty data
  • Nominatim API - To convert city names into geographic coordinates used in OpenAQ API query
  • Supabase - For getting and posting user comments
  • Chart.js - Javascript library to visualize PM2.5 trends with interactive graphs
  • noUiSlider - Javascript library to create simple date range slider
  • HTML/CSS/Javascript - For building the app!
